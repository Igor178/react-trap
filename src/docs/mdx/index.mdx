# Getting started

## Install

```
npm install react-trap
```
## Import
```
import Trap from 'react-trap'
```
Or `useTrap` hook
```
import { useTrap } from 'react-trap'
```

## Usage 

<Playground >
  <Trap event="click">
    {(trapped, ref) => <Box innerRef={ref} isFocused={trapped} />}
  </Trap>
</Playground>

## How is Box looks like?

```
const style = isFocused => ({
  display: "flex",
  padding: "25px",
  justifyContent: "center",
  color: "#fff",
  backgroundColor: isFocused ? "#4caf50" : "#e91e63",
  userSelect: "none",
  cursor: "pointer"
});

const Box = ({ isFocused, innerRef }) => (
  <div ref={innerRef} style={style(isFocused)}>
    {isFocused ? "I'm focused" : "I'm out of focus"}
  </div>
);
```
